## iOS Setup (iPhone)

DriverTraxxx is a Progressive Web App — **no App Store download needed**. Follow these steps to install it on your iPhone home screen so it opens like a native app.

---

> ⚠️ **Must use Safari.** Chrome and Firefox do not support home screen installation on iPhone.

Open Safari and go to:

```

```

## Using the App

---

### Logging a Vehicle Entry

From the **New Entry** tab, scan the vehicle barcode or enter the Serial ID manually, fill in the fields, and tap **Save Record**.

---

#### Scanning a Barcode

```
┌─────────────────────────────────┐
│                                 │
│   ┌─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─┐   │
│   │                         │   │
│   │   Point camera at tag   │   │
│   │  ─────────────────────  │   │
│   │   ← laser scan line →   │   │
│   │  ─────────────────────  │   │
│   │                         │   │
│   └─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─┘   │
│                                 │
│   [ 🔦 Torch ]    [ Cancel ]   │
└─────────────────────────────────┘
```

1. Tap the large green **SCAN BARCODE** button
2. Point your rear camera at the barcode on the vehicle tag
3. The phone **vibrates** and the Serial ID fills in automatically
4. The scanner closes and returns to the form

> 💡 Use the **torch button** for low-light conditions on the lot. The scanner reads **Code 39 and Code 128** — the formats used on Enterprise tags.

---

#### Manual Entry

1. Tap **Enter Manually** below the scan button
2. Type the Serial ID — it converts to uppercase automatically

---

### Status & Destination

Status and Destination sit **side by side** on the same line. Status is **required**.

```
┌─────────────────────────────────┐
│  STATUS          DESTINATION    │
│  ┌──────────┐   ┌──────────┐   │
│  │ CLEAN  ▼ │   │ GARAGE ▼ │   │
│  └──────────┘   └──────────┘   │
└─────────────────────────────────┘
```

| Status | Description |
|--------|-------------|
| CLEAN | Vehicle is clean and ready |
| DIRTY | Vehicle needs cleaning |
| PM | Preventive maintenance |
| MK | Make ready |
| MR | Major repair |
| AUDIT | Audit required |
| WI/DELETE | Write-in / Delete |
| GLASS | Glass damage |
| TI | Tire issue (shows tire selector) |
| OTHER | Other / miscellaneous |

| Option | Active Color | Behavior |
|--------|-------------|----------|

### Records & Filtering

The Records tab shows all entries. Use the filter bar to search by Serial ID, Status, Destination, No Tag flag, or date range. Tap any record card to open the detail view with GPS map, VIN info, and delete option.

---

### Data & Backup

Access **Data & Backup** from the hamburger menu.

| Action | Description |
|--------|-------------|
| **Auto Backup** | Runs every 30 minutes automatically |
| **Back Up Now** | Saves an immediate snapshot |
| **Export All Data** | Downloads a dated JSON file |
| **Import Data** | Merges an exported file onto this device |
| **Restore from Backup** | Overwrites current records from last backup |

> 💡 Export your data regularly and save the file to **Google Drive** or **iCloud** to protect against device loss or theft.

## Quick Tips

- ✅ Set your **Profile** before your first shift
- ✅ Add the app to your **home screen** for a full-screen native experience
- ✅ **Shuttle** and **Transport** checkboxes remember your last setting between entries
- ✅ Tap any day row in the **Dashboard** to jump directly to those records
- ✅ Use the **torch button** in the scanner for dark conditions on the lot
- ✅ The **performance banner** (avg trip time / cars per hour) appears after 2+ entries in a shift
- ✅ Export your data **weekly** to Google Drive or iCloud
- ✅ If the scanner struggles with a tag, use **Enter Manually** as a fallback

## Troubleshooting

### Buttons Not Responding

Clear your Safari or Chrome cache and reload the page. If you installed it on your home screen, delete the icon and re-add it after any app update on GitHub.

### GPS Not Working

- **iOS:** Settings → Privacy & Security → Location Services → Safari → While Using the App
- **Android:** Settings → Apps → Chrome → Permissions → Location → Allow

### Scanner Not Opening

Check camera permissions in your phone settings.

- **iOS:** Settings → Safari → Camera → Allow
- **Android:** Settings → Apps → Chrome → Permissions → Camera → Allow

### Records Missing After Update

Records are stored locally on your device. Updating the app file on GitHub does not affect your saved records. Go to **Data & Backup** in the menu and tap **Restore from Last Backup**.

### App Not Loading

Check your internet connection. The app works **offline** once loaded — your records stay safe on your device. GitHub Pages may occasionally be slow to load on first visit.
